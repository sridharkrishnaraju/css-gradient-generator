# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="gradient-generator.js"></script>
<gradient-generator></gradient-generator>
```

## React

```jsx
import "@sgbp/gradient-generator";
export default function Page() { return <gradient-generator />; }
```

## Vue

```vue
<script setup>
import "@sgbp/gradient-generator";
</script>

<template>
  <gradient-generator />
</template>
```

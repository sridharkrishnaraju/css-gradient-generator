# CSS Gradient Generator

A visual CSS gradient generator: pick linear or radial, set the angle and two colours, preview live, and copy the CSS. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="gradient-generator.js"></script>
<gradient-generator></gradient-generator>
```

## What it does

A CSS gradient is a smooth blend between colours, drawn by the browser with no image file. `linear-gradient()` runs along a line at a chosen angle; `radial-gradient()` spreads outward from a centre point. Because they are pure CSS, they stay sharp at any size and add zero network requests.

## When you'd use it

Building a hero background, a button, a card or an overlay and wanting copy-ready gradient CSS instead of exporting an image.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
